A story about a chiropractor,
LLMs and nine days

The day in the life of a medical professional

My wife is a chiro, and after every patient she turns the conversation into a SOAP note, trying to remember all the intricate details from memory.

Mentally taxing

Recalling every detail of an appointment (subjective, objective, assessment, plan), visit after visit, is exhausting.

Time-consuming

The writing happens after the clinical work is done. It eats into the day, then into the evening.

Not optional

Notes are legally required. In an audit, a gap in the record is a real liability, so they can't be skipped or rushed.

Obviously, AI is the answer,
but…

Why a local LLM?

I.Privacy is the premise

Protected Health Information (PHI) carries legal and ethical weight; cloud dictation and hosted LLM APIs were off the table from day one. On-device was the constraint everything else had to fit.

II.And an excuse to learn

How to run local LLMs in a real use case, how to build a native Mac app, and simply to tinker and play.

What didn't

postmortem.log — nine days, three lessons
✗

AI is persuasive when it's wrong

Days chasing a blocker in mlx-swift — the fix lived in mlx-swift-lm all along. Confident, well-argued, mistaken.

└─ fix: a separate agent, separate context, briefed to challenge the work

✗

A stale doc is a standing order

Our rules file still cited a deleted lint rule — the agent obeyed it anyway. It won't squint at the system of record like you do.

└─ fix: docs update in the same PR as the code they describe

✗

Native UI still needs human eyes

"Cancel" quit the entire app. CI, tests and two AI reviewers all passed it — five minutes of actually using the app caught it.

└─ fix: a human smoke-test on real hardware

$ fix · write it into the house rules · repeat

What I learned

1

Write the house rules down

AGENTS.md is a contract. The agent follows what's written, not what you meant. Investment in rules paid back every session.

2

Make the AI test its own work

The test suite is what turns "plausible" into "true". Never accept agent code without the test that would catch its lie.

3

A second AI reviewer isn't a gimmick

Writer and reviewer from different vendors disagree in useful ways. The disagreement is the value.

4

Small PRs beat clever prompts

The 23-minute merge loop did more for quality than any prompt engineering. Keep the unit of work reviewable.
